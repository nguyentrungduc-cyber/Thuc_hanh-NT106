# 📘 Lab 02 – File and Stream I/O in C#

## 📖 Giới thiệu

Lab này tập trung vào việc làm quen với **File và Stream I/O trong C#**, giúp sinh viên hiểu cách đọc/ghi dữ liệu từ file và xử lý dữ liệu bằng luồng (stream).

---

## 🎯 Mục tiêu

* Hiểu cách làm việc với file trong C#
* Sử dụng các lớp như:

  * `File`
  * `FileStream`
  * `StreamReader`
  * `StreamWriter`
* Thực hành đọc và ghi dữ liệu từ file
* Xử lý dữ liệu dạng text và binary

---

## 🛠️ Nội dung chính

### 🔹 1. Làm việc với File

* Tạo file
* Xóa file
* Kiểm tra file tồn tại

### 🔹 2. FileStream

* Mở file bằng `FileStream`
* Đọc/ghi dữ liệu dạng byte

### 🔹 3. StreamReader / StreamWriter

* Đọc file text
* Ghi file text
* Xử lý dữ liệu dòng

### 🔹 4. Binary Reader / Writer

* Làm việc với dữ liệu nhị phân
